---

### Tabela para gerenciar dados de pacientes de um nutricionista, o usuário é capaz de:
- Adicionar paciente
- Editar dados dos pacientes
- Deletar pacientes
- Filtrar pacientes por nome, peso, altura e imc
- Ordenar pacientes por ordem alfabética
- Visualizar dados mais completos do paciente, como saber se o paciente está acima do peso ou não, ou mesmo qual o seu peso ideal

### Esse projeto conta com:
- O uso de ReactJS e CSS puro
- O uso de context api para compartilhar estado entre os componentes
- Lógicas de componentes modais para mostrar informações na tela, assim seguindo o conceito de SPA
